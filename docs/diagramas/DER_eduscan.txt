================================================================================
                            RELACIONES (FK)
================================================================================

  evaluacion.id_alumno  →  alumnos.id_alumno          (1 a N)
  respuesta.id_evaluacion → evaluacion.id_evaluacion  (1 a N)
  respuesta.id_pregunta → pregunta.id_pregunta        (1 a N)

================================================================================
                            TIPOS DE DATOS
================================================================================

  INT(11)      → Entero de hasta 11 dígitos
  VARCHAR(100) → Texto variable hasta 100 caracteres
  DECIMAL(5,2) → Número decimal con 2 decimales (ej: 4.50)
  DATE         → Fecha (YYYY-MM-DD)
  BOOLEAN      → Verdadero/Falso (TRUE/FALSE)
  TEXT         → Texto largo (hasta 65,535 caracteres)
