# Historias de Usuario - EduScan

## Usuarios del Sistema

| Usuario | Rol | Descripción |
|---------|-----|-------------|
| Docente | Profesor | Corrige exámenes, ve estadísticas, gestiona alumnos |
| Administrador | Admin | Gestiona docentes, configura el sistema |
| Alumno | Estudiante | (Futuro) Ve sus calificaciones |


## Historias de Usuario - Docente

1. Como docente, quiero tomar una foto de un examen con mi celular para que el sistema lo corrija automáticamente.

2. Como docente, quiero subir una imagen desde mi galería para corregir exámenes que ya tengo guardados.

3. Como docente, quiero ver la nota final del examen (0.0 a 5.0) para saber el rendimiento del alumno.

4. Como docente, quiero ver cuántas preguntas correctas e incorrectas tuvo el alumno para identificar áreas de mejora.

5. Como docente, quiero guardar la calificación automáticamente en la base de datos para llevar un historial.

6. Como docente, quiero ver un dashboard con estadísticas generales (promedio, mejor nota, top alumnos) para evaluar el rendimiento grupal.

7. Como docente, quiero ver la lista de todos mis alumnos para tener un registro actualizado.

8. Como docente, quiero agregar, editar y eliminar alumnos para mantener la base de datos actualizada.

9. Como docente, quiero gestionar módulos (materias) para organizar los exámenes por área.

10. Como docente, quiero gestionar exámenes para asociarlos a módulos y fechas específicas.

11. Como docente, quiero ver el historial de puntajes de cada alumno para hacer seguimiento de su progreso.

12. Como docente, quiero comparar el rendimiento de un alumno por módulo para identificar fortalezas y debilidades.

13. Como docente, quiero exportar los datos de alumnos a CSV para tener un respaldo o compartirlos.

14. Como docente, quiero acceder desde mi celular o computador para usar la plataforma en cualquier momento.

15. Como docente, quiero que la IA analice las respuestas del examen para no tener que corregir manualmente.


## Historias de Usuario - Administrador

16. Como administrador, quiero gestionar los usuarios docentes para controlar quién accede al sistema.

17. Como administrador, quiero ver los logs de actividad para auditar el uso del sistema.

18. Como administrador, quiero configurar los parámetros de la IA (umbrales de aprobación) para ajustarlos a las necesidades.

19. Como administrador, quiero respaldar la base de datos para prevenir pérdida de información.


## Historias de Usuario - Alumno (Futuro)

20. Como alumno, quiero ver mis calificaciones para conocer mi rendimiento.

21. Como alumno, quiero ver mis respuestas correctas e incorrectas para aprender de mis errores.
